# **App Name**: MacroMentor

## Core Features:

- User Authentication: Secure user registration and login functionality for personalized profiles.
- AI Macro Analysis: Users can upload food images to automatically estimate macronutrient content using an AI vision tool.
- Manual Meal Logging: Manually log meals by searching a comprehensive food database for macro information.
- Daily Macro Summary: View a daily dashboard showing total macros consumed and progress towards personal dietary goals.
- Profile Management: Manage personal profile settings including dietary goals, preferences, and progress tracking.

## Style Guidelines:

- Primary color: A vibrant, fresh green (#30E042), embodying vitality and growth.
- Background color: A subtle, light green (#EBF5ED), providing a clean and calming canvas.
- Accent color: A contrasting teal-green (#36DBAF), used for highlights and interactive elements to maintain a fresh aesthetic.
- Body and headline font: 'Inter' (sans-serif) for its modern, clean, and objective readability, suitable for data-rich interfaces.
- Use minimalist, line-art icons that are intuitive and consistent, reflecting food, nutrition, and data visualization themes.
- A clean and organized layout with a strong focus on readability, ensuring macro data is easily digestible and actionable at a glance, optimized for mobile-first interaction.
- Subtle and functional micro-animations for feedback on user actions (e.g., logging a meal, reaching a goal) and smooth transitions between dashboard views.
