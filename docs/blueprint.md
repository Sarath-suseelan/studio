# **App Name**: MealIQ

## Core Features:

- Intelligent Meal Logging: Allow users to input their meals using natural language (e.g., 'A bowl of oatmeal with a banana and two spoons of peanut butter') for automated macronutrient estimation using an AI tool.
- Food Item Database: Provide a searchable database of common food items with pre-calculated macronutrient values for manual entry or verification.
- Macronutrient Breakdown Display: Visually display the estimated macronutrient breakdown (carbs, proteins, fats) for each logged meal and daily totals.
- Meal History & Overview: Enable users to view a chronological history of their logged meals and see trends in their macronutrient intake over time.
- Nutritional Goal Setting: Allow users to set and track their daily macronutrient goals (carbohydrates, protein, fat).

## Style Guidelines:

- Primary color: A balanced and refreshing green, suggesting health and vitality. Hex: #63A16C
- Background color: An almost-white shade with a very subtle green tint for a clean, airy feel. Hex: #EEF3EF
- Accent color: A vibrant, clear lime green for highlights, interactive elements, and positive feedback. Hex: #ACE04D
- Main font: 'Inter' (sans-serif) for both headlines and body text, providing excellent readability and a modern, objective look essential for data presentation.
- Use clear, minimalistic line icons that are easily recognizable for food items, data metrics, and navigation, maintaining a clean and focused user experience.
- Employ a card-based layout for meal entries and data summaries, ensuring information is presented in digestible, scannable blocks with ample whitespace.
- Incorporate subtle transitions for navigation and data updates, offering a smooth and intuitive user flow without being distracting or delaying interactions.
